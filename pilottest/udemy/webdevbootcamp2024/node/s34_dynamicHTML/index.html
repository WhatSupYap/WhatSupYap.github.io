34. 템플레이팅으로 동적 HTML 구성하기

템플레이팅이란?

  • (내 생각) Template 템플릿에서 가져와서 템플릿화 한다는 의미로 받아들여짐
  • 동적 html을 만들기 위한 도구: EJS(Embedded Javascript)
  • Handlebars, Jade, Pug, Nunjucks등이 있다

EJS용 Express 구성하기

  • 서버 구성
    • npm init -y npm i express
  • index.js에 파일작성
  • views 폴더가 기본 디렉토리임

뷰 디렉토리 설정하기

// __dirname 혹은 소스가 있는 디렉토리 app.set('views', path.join(__dirname, '/views'));
